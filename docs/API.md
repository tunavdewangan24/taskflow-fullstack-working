# TaskFlow API Documentation

## Base URL
http://localhost:5000

## Common API Routes

### Get all tasks
GET /api/tasks

### Create a task
POST /api/tasks

### Update a task
PUT /api/tasks/:id

### Delete a task
DELETE /api/tasks/:id
